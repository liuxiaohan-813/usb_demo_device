## usb模拟鼠标流程
- 初始化
  - 配置当前usb为device，配置当前为全速模式，以及check SOF帧的cycle
  - 配置phy check connect的time
  - 配置使用的EP方向和fifo深度（这里只需要EP0来配置和EP1作为数据接收）
  - phy初始化：选择phy类型（外置的2.0的phy）、复位phy
